# NetSFC - Ubiquitous WiFi & POI Heatmap System

System for Keio University Shonan Fujisawa Campus (SFC) 

---

## Project Overview 

NetSFC is an interactive SFC map that shows POI and WiFi heatmap so that staffs, faculty members, and students can easily find their perfect study or work spot.
